# 线性表：顺序表，链表，栈，队列


## 顺序表（数组）
- 从头开始存，连续存储，不能跳跃。  
    缺陷：  
        1. 空间不够，需要增容。
        2. 避免频繁扩容，一般扩容2倍，可能浪费空间。
        3. 要求从数据开始位置连续存储，效率不高。
